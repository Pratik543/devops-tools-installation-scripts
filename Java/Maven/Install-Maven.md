# Maven Installation

> Official Documentation: https://maven.apache.org/install.html

> Note if java is not installed it will install automatically. If you want to install specific version of java then install it manually

# Ubuntu/Debian
```sh
sudo apt update
sudo apt install maven -y
```

# Amazon Linux
```sh
sudo yum install maven -y
```

# Fedora/RHEL/CentOS
```sh
sudo dnf install maven -y
```

# Verify Installation
```sh
mvn -version
```
